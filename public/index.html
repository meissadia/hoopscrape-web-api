Player

Endpoint

http://hoopscrape-api.herokuapp.com/v1/player/{player-id}

  • player-id - numeric player id

Sample Response

  "name": "Donovan Mitchell",
  "position": "SG",
  "college": "Louisville",
  "weight": "215",
  "h_ft": "6",
  "h_in": "3",
  "age": "21"
}
